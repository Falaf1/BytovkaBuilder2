запустите программу, после закройте ее, скопируйте содержимое папки Images в папке проект, перейдите по пути: BytovkaBuilder2-master\проект\bin\Debug\Images и загрузите туда фотографии, после снова запускайте программу и у вас будет работать предпросмотр бытовок по внешнему виду.

# Конструктор бытовки

Приложение для Windows Forms, которое помогает подобрать материалы для бытовки и сформировать техническое задание в Excel.

## Функции
- Выбор материалов внешней и внутренней отделки
- Выбор материалов для пола
- Настройка окон и дверей
- Просмотр внешнего вида бытовки
- Экспорт ТЗ в Excel

## Установка
1. Скачайте последний релиз
2. Распакуйте архив
3. Запустите `BytovkaBuilder.exe`

## Требования
- Windows 7 и выше
- .NET Framework 4.7.2
